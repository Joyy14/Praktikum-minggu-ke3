Project Unggulan Saya

Sistem Data Pengguna Admin Jula-Jula

Sistem ini merupakan aplikasi berbasis web yang dikembangkan untuk membantu admin dalam mengelola data pengguna Jula-Jula secara terstruktur dan efisien.

Aplikasi ini memiliki fitur utama seperti: pengelolaan data pengguna (CRUD), sistem login admin, validasi input data, pencarian data, serta integrasi database menggunakan MySQL.

Tujuan dari sistem ini adalah untuk meningkatkan efisiensi pengelolaan data serta meminimalisir kesalahan pencatatan manual.
